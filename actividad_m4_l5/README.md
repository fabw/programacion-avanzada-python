# README

## 1. ¿Qué tipo de error creo que es más común en programas reales?

Creo que uno de los errores más comunes en programas reales ocurre cuando el usuario ingresa datos incorrectos o en un formato que el programa no espera. Por ejemplo, escribir texto cuando se pide un número. Este tipo de situación puede generar errores como `ValueError` y afectar el funcionamiento del programa si no se controla bien.

## 2. ¿Por qué es importante manejar excepciones?

Manejar excepciones es importante porque evita que el programa se cierre de forma repentina cuando ocurre un error. Además, permite mostrar mensajes más claros al usuario, ayuda a identificar problemas con mayor facilidad y hace que el programa sea más estable y fácil de mantener.
